# 🏦 نظام طلبات التحويل البنكي الإلكتروني - جمعية كاريتاس مصر

نظام ويب تفاعلي ذكي مصمم لتسهيل وتبسيط إجراءات إصدار طلبات التحويلات البنكية الرسمية لجمعية كاريتاس مصر، مع تقليل الأخطاء البشرية لضمان توافق الطلبات مع المعايير المصرفية.

## 🌟 أبرز المميزات (Features)

* **🔄 نظام ديناميكي ثنائي الأوضاع:** إمكانية التبديل الفوري بين (التحويلات الداخلية بين حسابات الجمعية) و(التحويلات لجهات خارجية)، مع قفل الحقول غير المطلوبة تلقائياً بناءً على اختيارك لمنع التشتيت.
* **✍️ تفقيط تلقائي لحظي (Auto-Tafqeet):** تحويل المبالغ المالية من أرقام إلى حروف لغة عربية رصينة فوراً أثناء الكتابة لمنع اختلاف التدوين.
* **⚠️ نظام التحقق الذكي من البيانات (Form Validation):** فحص شامل لجميع الحقول الإجبارية قبل السماح بالطباعة، وعرض شريط تحذيري بالحقوق الناقصة مع تحديدها باللون الأحمر.
* **📅 تلميحات تواريخ مرنة:** استنتاج اسم اليوم تلقائياً بمجرد اختيار التاريخ للتأكد من سلامة التوقيتات الإدارية.
* **🖨️ هندسة طباعة ذكية (A4 Optimized):** إعدادات طباعة متقدمة تخفي لوحات التحكم والأزرار عند الطباعة أو الحفظ كـ PDF، مما ينتج مستنداً رسمياً مطابقاً للأوراق الرسمية للمؤسسة.
* **🎨 واجهة مستخدم بهوية مؤسسية:** استخدام ألوان وتنسيقات تعكس الهوية البصرية الرسمية لجمعية كاريتاس مع دعم كامل للشاشات المختلفة والهواتف المحمولة.

## 🛠️ التكنولوجيات المستخدمة (Technologies)

* **HTML5:** لبناء الهيكل والأقسام والنموذج بشكل دلالي متوافق.
* **CSS3:** لتصميم الواجهة العصرية، مع استخدام تقنية `@media print` لضبط جودة الورق المطبوع.
* **JavaScript (Vanilla JS):** لبناء المنطق التفاعلي (التفقيط، التحقق من البيانات، ديناميكية الحقول وتغيير التبويبات).

## 🚀 طريقة التشغيل والاستخدام

1. قم بتحميل الملف الرئيسي `index.html` أو تصفحه مباشرة عبر خاصية **GitHub Pages**.
2. اختر نوع التحويل المطلوب من التبويب العلوي (داخلي / خارجي).
3. قم بملء الحقول المطلوبة، وسيظهر لك تفقيط المبلغ فوراً أسفل خانة القيمة.
4. اضغط على زر **🖨️ طباعة / حفظ PDF** لحفظ المستند بشكل رسمي أو طباعته مباشرة لتوقيعه من قبل الإدارة المالية (يحتوي المستند على مربعات التوقيعات الرسمية الجاهزة).
5. لتفريغ كافة البيانات والبدء من جديد، استخدم زر **تفريغ النموذج**.

## 📌 ملاحظات التطوير
* المشروع يعمل بالكامل على المتصفح (Client-side) دون الحاجة لخوادم أو قواعد بيانات لضمان السرعة وسرية البيانات المالية المدخلة أثناء الاستخدام اليومي في الفروع.
